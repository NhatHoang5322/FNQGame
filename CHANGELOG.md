# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-10

### Added
- Initial release of FNQ Studio website
- User authentication system (register/login/logout)
- Bilingual support (Vietnamese and English)
- Game shop with demo and full version options
- Account dashboard with balance and transaction history
- Payment system with multiple payment methods
- QR code payment integration
- Responsive design for all devices
- LocalStorage-based user data persistence
- Comprehensive documentation and setup guides

### Features
- User registration with 100k VNĐ starting balance
- Game purchase system (50k VNĐ for full version)
- Balance top-up functionality
- Transaction history tracking
- Language switching (Vietnamese/English)
- Bilingual payment and recharge pages
- Game showcase for NEON QUEST and HARVESTAY
- Studio information and lore section
- Mobile-responsive design
- Dark theme with purple and cyan accents

### Technical
- Pure JavaScript (no frameworks)
- CSS with CSS variables for theming
- SVG support for game artwork
- QR code generation via CDN
- Modern web standards (HTML5, CSS3)
- Cross-browser compatibility
